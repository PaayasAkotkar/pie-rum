CLI-VERSION of PIE RUM

## Quick-Start
1. if on windows, 
run the install.bat 
or on mac or linux install.sh
2. run this in main.go example.PlayPIERUMCLIBasicExample()
3. check if the cli-installed correctly 
```text 
pie-rum --version
```
4. wait for the PIE RUM server to start than use the command 
```text 
pie-rum post-test main.yaml
```
